Octane benchmarks

Raw numbers

The experiment compares 5 configurations per benchmark:

  1. Default: Plain TypeScript compiler output
  2. Safe: Safe TypeScript compiler output (with instrumented runtime checks)
  3. Safe Optimized: Safe TypeScript compiler output, uses inlined runtime checks
  4. Weak Map: Safe TypeScript compiler output, uses a separate Weak Map to maintain RTTI tags
  5. TS*: Safe TypeScript compiler output, uses an eager tagging strategy
For each benchmark, we report arithmetic mean and margin of error for both typed and untyped versions of the benchmark. All times are in seconds.
